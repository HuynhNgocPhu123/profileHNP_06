Quản lý Nhân sự bằng Nhận diện Khuôn mặt

Vai trò: Thành viên nhóm (2 người)

Xây dựng hệ thống quản lý nhân sự tích hợp công nghệ nhận diện khuôn mặt để điểm danh, xác thực và quản lý ra vào.

  • Giao diện web sử dụng PHP, quản lý nhân sự, kế toán, quyền truy cập.
  • Backend nhận diện khuôn mặt bằng Python (Flask) sử dụng OpenCV.
  • Lưu trữ dữ liệu khuôn mặt và nhân sự trong MySQL; giao tiếp PHP – Python qua API.
  • Tự động cập nhật dữ liệu điểm danh khi nhận diện khuôn mặt thành công.
